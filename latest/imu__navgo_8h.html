Detailed Description

Interface for the IMU on the NavGo board.

Interface for the IMU on the Umarim board.

  • Gyroscope: Invensense ITG-3200
  • Accelerometer: Analog Devices ADXL345
  • Magnetometer: Honeywell HMC5883L
  • Gyroscope: Invensense ITG-3200
