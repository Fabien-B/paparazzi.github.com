Module configuration options

Define Options

  • name: BLOB_LOCATOR_CAMERA value: front_camera|bottom_camera
    Description: Video device to use
  • name: BLOB_LOCATOR_FPS value: 0
    Description: The (maximum) frequency to run the calculations at. If zero, it will max out at the camera frame rate

Module functions

Init Functions

These initialization functions are called once on startup.

  • cv_blob_locator_init()

Event Functions

These event functions are called in each cycle of the module event loop.

  • cv_blob_locator_event()

Periodic Functions

These functions are called periodically at the specified frequency from the module periodic loop.

  • cv_blob_locator_periodic()
    • Frequency in Hz: 30
    • Autorun: TRUE
      Periodic function automatically starts after init.
    • Start function: cv_blob_locator_start()
      Executed before the periodic function starts.
    • Stop function: cv_blob_locator_stop()
      Executed after the periodic function stops.
