cloud_sensor module

Get data from Cloud Sensor

  • compute Liquid Water Content (LWC) value from PAYLOAD_FLOAT data
  • get already computed LWC from PAYLOAD_COMMAND data

Example for airframe file

Add to your firmware section:

<module name="cloud_sensor"/>

Module functions

Init Functions

These initialization functions are called once on startup.

  • cloud_sensor_init()

Periodic Functions

These functions are called periodically at the specified frequency from the module periodic loop.

  • cloud_sensor_report()
    • Frequency in Hz: 2.
    • Autorun: TRUE
      Periodic function automatically starts after init.

Datalink Functions

Whenever the specified datalink message is received, the corresponing handler function is called.

  • on message PAYLOAD_COMMAND call LWC_callback(buf)
  • on message PAYLOAD_FLOAT call cloud_sensor_callback(buf)
