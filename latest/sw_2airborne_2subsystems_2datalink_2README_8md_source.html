22 HOWTO use the datalink library
23 ------------------------------
24 
25 1. Copy the folder `datalink` in the export directory to your project directory (or untar the file `pprz_datalink.tar`)
26 
27 2. In your project makefile:
28  * define the variable `PPRZ_DATALINK_PORT` to your device (ex: `PPRZ_DATALINK_PORT=XXX`)
29  * define the variable `PPRZ_DATALINK_DIR` to your extract folder
30  * include the library makefile `include ${PPRZ_DATALINK_DIR}/datalink/pprz_datalink.mk`
31  * add `PPRZ_DATALINK_CFLAGS` to your cflags
32  * add `PPRZ_DATALINK_SRCS` to your sources (if your project requires a list of object files, you can use `PPRZ_DATALINK_OBJS` instead)
33 
34 3. Device interface
35  You need to provide an interface to send the data. If your device is named `XXX`, you need to provide the following functions or macros
36  * `XXXCheckFreeSpace(len)` that return true if your device has enough space for `len` bytes
37  * `XXXTransmit(c)` that send a single byte `c` using the device `XXX` (or store them in a buffer)
38  * `XXXSendMessage()` that flushs the bytes to send (if needed)
39  * `XXXChAvailable()` that returns true new characters are available for reading
40  * `XXXGetch()` taht returns the next character to parse
41 
42  Usually, an UART interface is used for datalink.
43 
44 4. Message parsing and includes:
45  in a C file of you project (for example a file named `datalink.c`), add the following lines:
46  ~~~{.c}
47  #define DATALINK_C 1 // this should be placed before the includes
48 
49  #inlcude "uart.h" // or change this include to the header of your actual device interface
50  #include "downlink.h"
51 
52  void dl_parse_msg(void) {
53  <add your code to parse messages here>
54  }
55  ~~~
56  You also need to poll for new characters, which means that you have to call the `DatalinkEvent()` function in your main loop.
57  This function is non-blocking and calls the decoder on each received byte. Finally the `dl_parse_msg` function is called when a complete message is received (checksum is valid).

81 void dl_parse_msg(void) {
82  int msg_id = dl_buffer[1];
83 
84  switch (msg_id) {
85  case DL_PONG:
86  led2 =! led2; // toggle led on new PONG message
87  break;
88  default:
89  break;
90  }
