Digital Camera Triggering over UART link

Module XML file: digital_cam_uart.xml

Send attitude and other relevant data to a computer based photocamera after a shootphoto command is given. And in return get certain value back from the computerbased photocamera to be able to intercat with the flightplan.

Module configuration options

Define Options

  • name: DC_AUTOSHOOT_QUARTERSEC_PERIOD value: 6
    Description: quarter_second
  • name: DC_AUTOSHOOT_DISTANCE_INTERVAL value: 50
    Description: grid in meters
  • name: DC_SHOT_SYNC_SEND value: TRUE|FALSE
    Description: send DC_SHOT message when photo was taken (default: TRUE)
