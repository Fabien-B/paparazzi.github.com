Module functions

Init Functions

These initialization functions are called once on startup.

  • gsm_init()

Event Functions

These event functions are called in each cycle of the module event loop.

  • gsm_event()

Periodic Functions

These functions are called periodically at the specified frequency from the module periodic loop.

  • gsm_init_report()
    • Period in seconds: 60.
    • Delay: 0.95
      Integer to impose a sequence (between 0 and main_freq/function_freq)
    • Autorun: TRUE
      Periodic function automatically starts after init.
  • gsm_send_report()
    • Period in seconds: 60.
    • Autorun: FALSE
      Periodic function is started by user command.

Raw gsm.xml file:

<!DOCTYPE module SYSTEM "module.dtd">
<module name="gsm">
<doc>
<description>GSM module</description>
</doc>
<header>
<file name="gsm.h"/>
</header>
<init fun="gsm_init()"/>
<periodic fun="gsm_init_report()" period="60." delay="0.95" autorun="TRUE"/>
<periodic fun="gsm_send_report()" period="60." autorun="FALSE"/>
<event fun="gsm_event()"/>
<makefile>
<file name="gsm.c"/>
</makefile>
</module>
