Detailed Description

Specific RAM section for DMA usage on F7

F1 ram0: 64ko std

F3 ram4: 8ko ccm, fast, no dma ram0: 40Ko std F37 ram0: 32Ko std F4 ram4: 64ko ccm, fast, no dma ram0: 128Ko std

F7 ram0: std, fast, no dma ram3: dma
